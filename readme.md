# ジグソーパズルの描画

## 動かし方

### 動作環境
十分条件なので、必ずしも合わせる必要はないです。
```
Windows 10 Pro
Python 3.6.10
Pygame 1.9.2
```

### 依存パッケージ
```
pygame
```

### 実行方法

```
python puzzle.py input.txt
```

## input

- $H$ : 盤面高さ
- $W$ : 盤面幅
- $A_{ij}$ : ピース番号（0が空白）
- $S$ : 操作列（空白の移動方向LURD）
  - 'L'なら空白を左のマスに移動させることを表す。
  - 盤外に空白を移動させるような不正な操作列は無いものとする。

### フォーマット

```
H W
A_0_0 ... A_0_{W-1}
...
A_{H-1}_0 ... A_{W-1}_{H-1}
S
```

### 具体例

```text
3 4
0 2 3 9
5 6 7 10
8 4 1 11
RRRDDD
```

## output

### 描画するもの

- コマ送りの自動・手動フラグ
- コマ送りの速度
- 盤面
- 各マスについて、ゴールまでの距離によって配色を変更

### 画面制御

- 画面右上の×ボタン、または、ESCAPEキーで終了
- 十字キーの下ボタンで、コマ送りの「再生・停止」を切り替え、上ボタンで「逆再生・停止」を切り替え
- 手動の場合は、十字キーの左右で、コマ送り
- 自動の場合は、ASDFキーでコマ送りの速度を切り替え（A: 遅い --> F: 速い、デフォルトは速い）
